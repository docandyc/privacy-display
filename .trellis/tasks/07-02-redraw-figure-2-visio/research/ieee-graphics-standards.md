# IEEE graphics constraints used for Figure 2

- IEEE two-column graphics width: 7.16 in / 182 mm / 43 picas.
- Prefer vector formats for rescaling; accepted vector submission formats include PDF, EPS, and PS.
- Non-vector color and grayscale graphics should exceed 300 dpi; black-and-white line art should exceed 600 dpi.
- IEEE recommends Helvetica, Times New Roman, Arial, Cambria, and Symbol.
- Type should appear approximately 9–10 pt at full size and remain consistent across figures.
- Color encodings should also use shape, line style, labels, or brightness contrast so the figure remains interpretable in grayscale.
